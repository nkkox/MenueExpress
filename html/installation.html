Installationsvorgänge

Ziel

Zwei Computer werden mit Ubuntu 16.04.7 LTS installiert: Einer dient als Server, der andere als Client. Beide benötigen NFS (Dateiserver) und NIS (Anmeldeserver).

1. Vorbereitung

  • USB-Stick: Installationsdatei von Ubuntu 16.04.7 LTS.
  • Speicherplatz: Mindestens 5 GB frei.
  • Backup: Sicherung wichtiger Dateien.

    • 2. Installation von Ubuntu

    - USB-Stick einstecken und Computer einschalten
    - Boot-Menü aufrufen (z.B. F12, Esc, F9) und USB-Stick auswählen.
    - Sprache wählen und Installation starten.
    - Updates und optionale Software installieren.
    - Festplatte formatieren und Installation bestätigen.
    - Standort und Tastaturlayout wählen.
    - Benutzerdaten eingeben:
  • Server: Name, "server", Benutzername, Passwort.
  • Client: Name, "client", Benutzername, Passwort.
    • - Optionen:Passwort zur Anmeldung erforderlich.
    - Installation abschließen und Computer neu starten.

    3. Installation von NFS

    NFS-Server:
  • Root-User: Passwort festlegen (sudo passwd root) und anmelden (su).
  • Installation: apt-get -y install nfs-kernel-server.

    • NFS-Client:
  • Root-User: Anmelden (su).
  • Installation: apt-get -y install nfs-common.
  • Automount: apt-get -y install autofs.

    • 4. Installation von NIS

    NIS-Server:
  • Anmelden (su).
  • Installation: apt-get -y install nis.

    • NIS-Client:
  • Root-User: Anmelden (su).
  • Installation: apt-get -y install nis.
